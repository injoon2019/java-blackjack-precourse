# 우아한테크코스 3기 프리코스 오프라인 코딩테스트 대비 
### 블랙잭 미션
- 블랙잭 게임을 진행하는 프로그램을 구현한다. 블랙잭 게임은 딜러와 플레이어 중 카드의 합이 21 또는 21에 가장 가까운 숫자를 가지는 쪽이 이기는 게임이다.


<br>

## 🚀 기능 요구사항
    
1. 게임에 참여할 사람의 이름을 입력받는다.
2. 이름은 쉼표로 분리한다. 
3. 사람 별로 배팅 금액을 입력 받는다. 
4. 딜러를 포함하여 카드를 두 장씩 나눠 준다.
5. 사람들이 가지고 있는 카드를 출력한다. 
    1. Ace는 1 또는 11로 계산할 수 있으며, King, Queen, Jack은 각각 10으로 계산한다.
    2. 처음 두 장의 카드 합이 블랙잭이 되면 베팅 금액의 1.5배를 딜러에게 받는다. 
    3. 딜러와 플레이어가 모두 동시에 블랙잭인 경우 베팅한 금액을 돌려 받는다. 
6. 추가로 카드를 받을지 물어본다. 
    1. 추가로 받는다하면 계속 허용한다.
    2. 단 21이 넘어가면 그 사람은 베팅 금액 모두를 잃게 된다. 
7. 결과를 출력한다.
8. 수익을 계산한다.
9. 수익을 출력한다.
